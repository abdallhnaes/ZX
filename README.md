# تطبيق إدارة بسطة محروقات بسيط

تطبيق Android بسيط جدًا يعمل بدون إنترنت لإدارة بسطة محروقات.

## الواجهات

1. تعيين القيم الابتدائية
   - الكاش الموجود
   - مخزون المازوت والبنزين والغاز
   - سعر البيع والشراء لكل مادة

2. واجهة البيع
   - بيع نقداً أو دين
   - تحديث المخزون والكاش تلقائياً
   - إضافة دين للزبون عند البيع بالدين

3. واجهة شراء الموارد
   - شراء نقداً أو دين علينا
   - تحديث المخزون والكاش تلقائياً
   - إضافة دين للمورد عند الشراء بالدين

4. واجهة الدين لنا وعلينا
   - عرض الديون لنا من الزبائن
   - عرض الديون علينا للموردين
   - تسجيل دفعات وتسديدات

## بناء APK عبر GitHub

ارفع الملفات على GitHub ثم شغّل:

Actions -> Build Fuel Stand Simple APK -> Run workflow

بعد انتهاء البناء ستجد APK في Artifacts باسم:

FuelStandSimple-debug-apk

## ملف الواجهة

يمكن تعديل التصميم والمنطق من:

app/src/main/assets/index.html
